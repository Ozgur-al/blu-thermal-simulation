  <action>
Create `thermal_sim/ui/results_tab.py` with a `ResultsSummaryWidget(QWidget)` class:

**Layout** (vertical, three sections with group boxes):

1. **Per-Layer Stats** (`QGroupBox("Layer Statistics")`):
   - `QTableWidget` with columns: Layer, T_max [C], T_avg [C], T_min [C], DeltaT [C]
   - One row per layer
   - `QHeaderView.ResizeMode.Stretch` for all columns
   - Read-only (no editing)

2. **Hotspot Ranking** (`QGroupBox("Top Hotspots")`):
   - `QTableWidget` with columns: Rank, Layer, X [mm], Y [mm], Temperature [C]
   - Up to 10 rows (configurable via constructor parameter `max_hotspots=10`)
   - Read-only
   - `cellClicked` signal connected to internal handler that emits custom `hotspot_clicked` signal

3. **Probe Readings** (`QGroupBox("Probe Readings")`):
   - `QTableWidget` with columns: Probe, Layer, X [mm], Y [mm], Temperature [C]
   - One row per probe
   - Read-only

**Signals:**
```python
hotspot_clicked = Signal(int, str, float, float)  # rank (1-based), layer_name, x_m, y_m
```

**Public API:**
```python
def update_data(
    self,
    layer_stats_data: list[dict],      # from layer_stats()
    hotspots: list[dict],               # from top_n_hottest_cells()
    probe_values: dict[str, float],     # {name: temp_c}
    probes: list,                       # Probe objects for location info
) -> None:
    """Populate all three tables with new simulation results."""
```

The `update_data` method:
- Clears and repopulates the layer stats table (one row per dict in `layer_stats_data`)
- Clears and repopulates the hotspot table (convert x_m/y_m to mm with `* 1000` for display)
- Clears and repopulates the probe table (match probe name to probe object for layer/location; convert x/y to mm for display)
- Stores `hotspots` list internally as `self._current_hotspots` for click-to-navigate

The `_on_hotspot_cell_clicked(row, col)` slot:
- Looks up `self._current_hotspots[row]`
- Emits `hotspot_clicked.emit(row + 1, hotspot["layer"], hotspot["x_m"], hotspot["y_m"])`

Format all temperature values to 2 decimal places. Format locations (mm) to 1 decimal place.

Use `QTableWidget.setEditTriggers(QTableWidget.EditTrigger.NoEditTriggers)` to make tables read-only.
Use `QTableWidget.setSelectionBehavior(QTableWidget.SelectionBehavior.SelectRows)` for the hotspot table to make row selection intuitive.
Use `QTableWidget.setAlternatingRowColors(True)` for readability.
