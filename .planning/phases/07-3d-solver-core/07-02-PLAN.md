**1. Add NodeLayout dataclass** (top of file, after imports):
```python
@dataclass(frozen=True)
class NodeLayout:
    nx: int
    ny: int
    n_per_layer: int          # nx * ny
    n_layers: int
    layer_offsets: tuple[int, ...]  # layer_offsets[l] = first node index for layer l

    def node(self, layer_idx: int, ix: int, iy: int, iz: int = 0) -> int:
        return self.layer_offsets[layer_idx] + iz * self.n_per_layer + iy * self.nx + ix

    @property
    def n_nodes(self) -> int:
        return self.layer_offsets[-1] + self.n_per_layer
```
For Phase 7 (nz=1 everywhere), `layer_offsets = (0, n_per_layer, 2*n_per_layer, ...)`. The `iz` parameter defaults to 0 and is unused until Phase 8.

**2. Update ThermalNetwork** to hold a `NodeLayout`:
- Add `layout: NodeLayout` field to ThermalNetwork
- Change `n_nodes` property to use `self.layout.n_nodes`
- Keep `n_layers` and `grid` for backward compatibility with solvers

**3. Update `_add_link_vectorized`** to accept `float | np.ndarray`:
```python
def _add_link_vectorized(n1, n2, conductance):
    """conductance: float or 1D ndarray matching len(n1)"""
    if isinstance(conductance, (int, float)):
        if conductance <= 0.0:
            return
        g = np.full(len(n1), conductance, dtype=float)
    else:
        g = np.asarray(conductance, dtype=float)
        # Filter out zero-conductance links
        mask = g > 0.0
        if not mask.all():
            n1, n2, g = n1[mask], n2[mask], g[mask]
        if len(g) == 0:
            return
    # ... rest unchanged (4 COO entries)
```

**4. Add zone rasterization function** (module-level helper):
```python
def _rasterize_zones(layer, project, grid, materials_dict) -> tuple[np.ndarray, np.ndarray, np.ndarray]:
    """Rasterize zones to per-cell material property arrays.

    Returns (k_in_plane_map, k_through_map, density_cp_map) each shape (ny, nx).
    - k_in_plane_map: in-plane conductivity per cell
    - k_through_map: through-plane conductivity per cell
    - density_cp_map: density * specific_heat per cell (for thermal capacity)
    """
```
Implementation details:
- If `layer.zones` is empty, auto-generate a full-coverage zone from `layer.material` (per locked decision).
- If `"Air Gap"` not in `materials_dict`, inject it from builtin values (k=0.026, k_through=0.026, density=1.2, cp=1005, emissivity=0.5). Use a local copy of materials_dict — NEVER mutate the project.
- Initialize material index array `mat_idx = np.zeros((ny, nx), dtype=int)` pointing to air (index 0).
- For each zone in order (last-defined wins), rasterize using AABB cell-overlap (same pattern as `_source_mask` for rectangles):
  ```python
  sx0 = zone.x - zone.width / 2.0
  sx1 = zone.x + zone.width / 2.0
  sy0 = zone.y - zone.height / 2.0
  sy1 = zone.y + zone.height / 2.0
  mask = (xx + half_dx > sx0) & (xx - half_dx < sx1) & (yy + half_dy > sy0) & (yy - half_dy < sy1)
  ```
- Zones that extend beyond grid bounds are clipped naturally by the mask (no cells match outside). Emit `warnings.warn()` if zone extends fully outside bounds.
- Build material property arrays by indexing into a list of Material objects.

**5. Replace lateral conductance with per-cell harmonic-mean** in `build_thermal_network`:
- For each layer, call `_rasterize_zones()` to get per-cell k_in_plane and k_through maps.
- Compute per-cell conductance:
  ```python
  g_x_per_cell = k_in_plane_map * layer.thickness * grid.dy / grid.dx  # (ny, nx)
  g_y_per_cell = k_in_plane_map * layer.thickness * grid.dx / grid.dy  # (ny, nx)
  ```
- Compute harmonic mean for x-links:
  ```python
  g_left  = g_x_per_cell[:, :-1]   # (ny, nx-1)
  g_right = g_x_per_cell[:, 1:]    # (ny, nx-1)
  g_x_link = 2.0 * g_left * g_right / (g_left + g_right)  # harmonic mean
  ```
- Compute harmonic mean for y-links similarly.
- Pass `g_x_link.ravel()` to `_add_link_vectorized` instead of the scalar `g_x`.
- **CRITICAL: For uniform layers (no zones, same material everywhere), harmonic mean of equal values = same value. The result is mathematically identical to the old scalar approach.**

**6. Update thermal capacity** computation to use per-cell material arrays:
```python
# Replace: c_node = material.density * material.specific_heat * layer.thickness * area
# With: per-cell capacity using density_cp_map from zone rasterization
density_cp_map = density_map * cp_map  # (ny, nx)
c_per_cell = density_cp_map * layer.thickness * grid.cell_area  # (ny, nx)
c_vec[start:start+n_per_layer] = c_per_cell.ravel()
```

**7. Update through-thickness conductance** to use per-cell k_through:
- For through-thickness links between adjacent layers, use the per-cell k_through values from each layer's zone rasterization.
- The resistance formula per cell pair becomes:
  ```python
  r_lower = lower.thickness / (2.0 * k_through_lower * area)  # per-cell array
  r_upper = upper.thickness / (2.0 * k_through_upper * area)  # per-cell array
  r_interface = lower.interface_resistance_to_next / area      # scalar, broadcast
  g_z = 1.0 / (r_lower + r_interface + r_upper)               # per-cell array
  ```
- Pass the per-cell array to `_add_link_vectorized`.

**8. Update top/bottom/side boundary conductance** to use per-cell k_through and emissivity:
- For top/bottom boundaries, compute per-cell surface conductance using the top/bottom layer's k_through map.
- For side boundaries, use per-cell k_in_plane values for perimeter cells.
- This is important for correctness when different materials are at the boundary.

**9. Replace node_index closure with layout.node():**
- Create `layout = NodeLayout(nx=grid.nx, ny=grid.ny, ...)` at start of `build_thermal_network`.
- Replace all `node_index(l, ix, iy)` calls with `layout.node(l, ix, iy)`.
- Pass `layout` to `_apply_heat_sources` (update its signature to accept `layout` parameter and use `layout.node()` internally).
- Store `layout` in ThermalNetwork.

**10. Update `build_heat_source_vector`** to use NodeLayout-compatible indexing.
The existing formula `layer_idx * n_per_layer + iy_arr * grid.nx + ix_arr` is identical to `layout.node()` for nz=1, so the math is unchanged. Just ensure it's consistent.

**Anti-patterns to avoid:**
- NO Python loops per cell for conductance — use vectorized NumPy array operations throughout
- Do NOT mutate `project.materials` — inject Air Gap into a local copy only
- Do NOT special-case uniform layers — harmonic mean of equal values naturally equals the same value
- Do NOT change the `_source_mask` function — it stays unchanged
