**Test 1: `test_zref05_single_layer_nz5_matches_1d_analytical`**

The test implements the physics from the RESEARCH.md ZREF-05 section:
- Single layer, 1x1 mesh (nx=1, ny=1), nz=5
- Material: k_through=k_in_plane=2.0 W/(m*K), density=1000, specific_heat=900, emissivity=0.9
- Layer thickness=0.005 (5 mm)
- Heat source: shape="full", z_position="distributed", power_w=3.0
- BCs: top h=10.0, bottom h=5.0, side h=0.0, T_amb=25.0, no radiation
- MeshConfig(nx=1, ny=1)

Build the 5-node analytical solution as a tridiagonal system:
- dz = thickness / nz = 0.001
- G_intra = k * area / dz (area = 0.10 * 0.10 = 0.01)
- G_bot = 1 / (dz/(2*k*area) + 1/(h_bot*area))
- G_top = 1 / (dz/(2*k*area) + 1/(h_top*area))
- q_node = Q_w / nz = 0.6 per node
- Build 5x5 A matrix and RHS, solve with np.linalg.solve

**Test 2: `test_zref03_interface_resistance_applies_at_layer_boundary`**

This test verifies ZREF-03: interface resistance applies ONLY at true layer boundaries, not within layers. Uses a 2-layer project, each with nz=3, with nonzero interface_resistance_to_next on the bottom layer.

Setup:
- 2 layers, 1x1 mesh (nx=1, ny=1)
- Layer 1 ("Bottom"): material "M", thickness=0.003, nz=3, interface_resistance_to_next=0.002 (m^2*K/W)
- Layer 2 ("Top"): material "M", thickness=0.003, nz=3, interface_resistance_to_next=0.0
- Material "M": k_through=k_in_plane=2.0, density=1000, specific_heat=900, emissivity=0.9
- Heat source: on "Bottom" layer, shape="full", z_position="bottom", power_w=2.0
- BCs: top h=10.0, bottom h=0.0 (adiabatic), side h=0.0, T_amb=25.0, no radiation
- MeshConfig(nx=1, ny=1)
- width=0.10, height=0.10

Build the 6-node analytical solution as a tridiagonal system:
- dz = 0.003 / 3 = 0.001 (same for both layers since same thickness and nz)
- area = 0.10 * 0.10 = 0.01
- G_intra = k * area / dz = 2.0 * 0.01 / 0.001 = 20.0 (within-layer links, NO interface resistance)
- G_inter = 1.0 / (dz/(2*k*area) + R_interface/area + dz/(2*k*area)) = 1.0 / (0.001/(2*2.0*0.01) + 0.002/0.01 + 0.001/(2*2.0*0.01)) = 1.0 / (0.025 + 0.2 + 0.025) = 1.0 / 0.25 = 4.0 (inter-layer link with interface resistance)
- G_top = 1.0 / (dz/(2*k*area) + 1/(h_top*area)) = 1.0 / (0.025 + 10.0) = 1.0 / 10.025 (top BC on node 5)
- Bottom BC: adiabatic (h=0), so G_bot = 0, no bottom BC link
- Heat: all power to node 0 (z_position="bottom"): q_node_0 = 2.0, q_nodes_1_to_5 = 0.0

Build 6x6 A matrix:
- Node 0 (bottom of layer 1): A[0,0] = G_intra, A[0,1] = -G_intra, rhs[0] = 2.0
- Node 1: A[1,0] = -G_intra, A[1,1] = 2*G_intra, A[1,2] = -G_intra, rhs[1] = 0
- Node 2 (top of layer 1): A[2,1] = -G_intra, A[2,2] = G_intra + G_inter, A[2,3] = -G_inter, rhs[2] = 0
- Node 3 (bottom of layer 2): A[3,2] = -G_inter, A[3,3] = G_inter + G_intra, A[3,4] = -G_intra, rhs[3] = 0
- Node 4: A[4,3] = -G_intra, A[4,4] = 2*G_intra, A[4,5] = -G_intra, rhs[4] = 0
- Node 5 (top of layer 2): A[5,4] = -G_intra, A[5,5] = G_intra + G_top, rhs[5] = G_top * T_amb

Solve with np.linalg.solve and compare to solver output.

Assert:
1. result.temperatures_c.shape == (6, 1, 1)
2. result.nz_per_layer == [3, 3]
3. result.z_offsets == [0, 3, 6]
4. For each z_idx in range(6): math.isclose(result.temperatures_c[z_idx, 0, 0], T_expected[z_idx], rel_tol=1e-9, abs_tol=1e-9)
5. Key physics check: the temperature jump at the layer boundary (T[2] - T[3]) should be significantly larger than within-layer jumps (T[0]-T[1], T[1]-T[2], etc.), confirming interface resistance is applied only at the boundary. Specifically, the heat flux Q through the interface is 2.0 W (all power flows upward since bottom is adiabatic). The temperature drop across the interface resistance alone is Q * R_interface / area = 2.0 * 0.002 / 0.01 = 0.4 K. Assert that T[2] - T[3] includes this 0.4 K interface contribution (it will also include the conduction half-thicknesses, so T[2] - T[3] should equal Q * (dz/(2*k*area) + R_interface/area + dz/(2*k*area)) = 2.0 * 0.25 = 0.5 K).

Mark with `pytest.mark.xfail(reason="ZREF-03: requires Plan 02 z-refinement in network builder", strict=False)`.
