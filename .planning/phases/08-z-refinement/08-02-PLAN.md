- If ndarray: use directly (must have same length as n1/n2)
- Update the zero-check: `if np.isscalar(conductance) and conductance <= 0.0: return`; for arrays, filter out zero-conductance pairs or skip the check (the vectorized assembly handles zeros)
```python
def _add_link_vectorized(n1: np.ndarray, n2: np.ndarray, conductance) -> None:

        if conductance <= 0.0:
            return

    else:
        g = np.asarray(conductance, dtype=float)
```

**7. Update INTER-LAYER z-z links (ZREF-03):**
Replace the existing through-thickness loop. The key change: use `dz_sub` (= thickness/nz) for the half-thickness terms, and connect the TOPMOST sublayer of layer l to the BOTTOMMOST sublayer of layer l+1:
```python

    lower = project.layers[l_idx]
    upper = project.layers[l_idx + 1]
    lower_mat = project.material_for_layer(l_idx)
    upper_mat = project.material_for_layer(l_idx + 1)
    dz_lower = lower.thickness / lower.nz

        (dz_lower / (2.0 * lower_mat.k_through * area))

```
