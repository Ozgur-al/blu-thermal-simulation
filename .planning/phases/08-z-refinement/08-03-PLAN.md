**Update all functions to use z_offsets for per-layer indexing.** The key pattern: where code currently does `temperature_map_c[idx]` to get a single layer, it must now do `temperature_map_c[z0:z1]` where z0=z_offsets[idx], z1=z_offsets[idx+1], and aggregate across z-sublayers.

**Helper function -- add at module level:**
```python
import logging
import warnings

logger = logging.getLogger(__name__)

def _z_offsets_for_result(result, n_layers: int) -> list[int]:
    """Return z_offsets from result or default to [0, 1, 2, ..., n_layers] for pre-z-refinement results."""
    if hasattr(result, 'z_offsets') and result.z_offsets is not None:
        return result.z_offsets
    return list(range(n_layers + 1))
```

**probe_temperatures(project, result):**
- Get z_offsets from result (or fallback: `_z_offsets_for_result(result, len(project.layers))`)
- For each probe, resolve z_node using probe.z_position:
```python
def _z_node_for_probe(probe, project, z_offsets):
    l_idx = project.layer_index(probe.layer)
    nz = z_offsets[l_idx + 1] - z_offsets[l_idx]
    z_base = z_offsets[l_idx]
    z_pos = getattr(probe, 'z_position', 'top')
    if z_pos == 'top':
        return z_base + nz - 1
    elif z_pos == 'bottom':
        return z_base
    elif z_pos == 'center':
        return z_base + nz // 2
    elif isinstance(z_pos, int):
        return z_base + min(z_pos, nz - 1)
    return z_base + nz - 1  # default top
```
- Replace `result.temperatures_c[l_idx, iy, ix]` with `result.temperatures_c[z_node, iy, ix]`

**probe_temperatures_over_time(project, result):**
- Same z_node resolution
- Replace `result.temperatures_time_c[:, l_idx, iy, ix]` with `result.temperatures_time_c[:, z_node, iy, ix]`

**layer_stats(temperature_map_c, layer_names, z_offsets=None):**

- If z_offsets is None, apply the fallback with a defensive warning for nz>1 results:

```python
    # Fallback: assume 1 z-node per layer (pre-z-refinement or nz=1 results).
    # WARNING: This fallback is incorrect for nz>1 results -- callers should

```
- Replace `temperature_map_c[idx]` with `temperature_map_c[z_offsets[idx]:z_offsets[idx+1]]` -- this gives [nz, ny, nx]
- Aggregate: `.max()`, `.mean()`, `.min()` naturally work across the z dimension

**_top_n_from_map(temperature_map_c, layer_names, dx, dy, n, z_offsets=None):**

- If z_offsets is None: `z_offsets = list(range(len(layer_names) + 1))`
- After computing `layer, rem = divmod(int(idx), nx * ny)`: the `layer` value is now a z-node index, not a physical layer index
- Map z-node to physical layer name: iterate z_offsets to find which layer the z-node belongs to:
```python
def _z_to_layer_idx(z_node: int, z_offsets: list[int]) -> int:
    for i in range(len(z_offsets) - 1):
        if z_offsets[i] <= z_node < z_offsets[i + 1]:
            return i
    return len(z_offsets) - 2  # fallback to last layer
```
- Use `layer_names[_z_to_layer_idx(layer, z_offsets)]` instead of `layer_names[layer]`

**csv_export.py -- export_temperature_map_array():**
The current function takes `temperature_map_c: np.ndarray` with shape [n_layers, ny, nx]. After Phase 8, the shape is [total_z, ny, nx]. Update to accept an optional `z_offsets` parameter and export per-physical-layer:
```python
def export_temperature_map_array(
    temperature_map_c: np.ndarray,
    layer_names: list[str],
    dx: float,
    dy: float,
    output_path: str | Path,
    z_offsets: list[int] | None = None,
) -> None:
    path = Path(output_path)
    path.parent.mkdir(parents=True, exist_ok=True)
    total_z, ny, nx = temperature_map_c.shape
    if z_offsets is None:
        z_offsets = list(range(len(layer_names) + 1))
    with path.open("w", newline="", encoding="utf-8") as f:
        writer = csv.writer(f)
        writer.writerow(["layer", "x_m", "y_m", "temperature_c"])
        for l_idx, layer_name in enumerate(layer_names):
            z0 = z_offsets[l_idx]
            z1 = z_offsets[l_idx + 1]
            # For CSV: export the top sublayer per layer (default visualization layer)
            # Per CONTEXT.md: "Default temperature map visualization shows top sublayer per layer"
            top_z = z1 - 1
            for iy in range(ny):
                y = (iy + 0.5) * dy
                for ix in range(nx):
                    x = (ix + 0.5) * dx
                    writer.writerow([layer_name, x, y, float(temperature_map_c[top_z, iy, ix])])
```

```python
def test_zref04_mixed_nz_project_solves_without_shape_error():
    """A project with nz=[1, 3, 2] solves correctly (ZREF-04).

    total_z = 1 + 3 + 2 = 6. When all nz=1, total_z would equal n_layers=3,
    so this test specifically validates that mixed nz values produce the
    correct expanded z-axis shape.
    """
    mat = Material("M", 2.0, 2.0, 1000.0, 900.0, 0.9)
    project = DisplayProject(
        name="Mixed nz",
        width=0.05,
        height=0.05,
        materials={"M": mat},
        layers=[
            Layer(name="L1", material="M", thickness=0.001, nz=1, interface_resistance_to_next=0.0),
            Layer(name="L2", material="M", thickness=0.003, nz=3, interface_resistance_to_next=0.0),
            Layer(name="L3", material="M", thickness=0.002, nz=2),
        ],
        heat_sources=[HeatSource(name="Q", layer="L1", power_w=1.0, shape="full")],
        boundaries=BoundaryConditions(
            top=SurfaceBoundary(ambient_c=25.0, convection_h=10.0, include_radiation=False),
            bottom=SurfaceBoundary(ambient_c=25.0, convection_h=5.0, include_radiation=False),
            side=SurfaceBoundary(ambient_c=25.0, convection_h=0.0, include_radiation=False),
        ),
        mesh=MeshConfig(nx=1, ny=1),
    )
    result = SteadyStateSolver().solve(project)
    total_z = 1 + 3 + 2  # = 6, NOT n_layers=3
    assert result.temperatures_c.shape == (total_z, 1, 1)
    assert result.nz_per_layer == [1, 3, 2]
    assert result.z_offsets == [0, 1, 4, 6]
    assert not np.any(np.isnan(result.temperatures_c))
    # Temperature should decrease from heat source (L1) toward top BC
    # L1 is bottommost with heat, so its temperature should be highest
```
