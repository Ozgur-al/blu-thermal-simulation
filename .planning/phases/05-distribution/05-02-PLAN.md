
    The script must:

    a. **Step 1 — Clean and build**: Run `subprocess.run(["pyinstaller", "--clean", "--noconfirm", "ThermalSim.spec"], check=True)`. Print clear step headers.

    b. **Step 2 — Copy examples**: Copy all 3 JSON files from `examples/` to `dist/ThermalSim/examples/` using `shutil.copy2`. Create the destination directory with `mkdir(exist_ok=True, parents=True)`.

    d. **Step 4 — Defender scan (optional)**: Find `MpCmdRun.exe` using the glob pattern from RESEARCH.md Pitfall 7:
       ```python
       candidates = sorted(glob.glob(r"C:\ProgramData\Microsoft\Windows Defender\Platform\*\MpCmdRun.exe"))
       mpcmdrun = candidates[-1] if candidates else r"C:\Program Files\Windows Defender\MpCmdRun.exe"
       ```
       Run custom scan (`-ScanType 3 -File dist/ThermalSim`). If MpCmdRun not found at all, warn and skip. If scan returns non-zero, print warning but do NOT fail the build (different environments may have different Defender configs).

    e. **Step 5 — Create zip**: Use `zipfile.ZipFile` to create `ThermalSim-v1.0.zip` (version from `thermal_sim.core.paths.APP_VERSION`). All files archived under `ThermalSim/` prefix so extraction creates a single folder.

    f. **Final output**: Print summary — zip path, size in MB, success/warning status.

    g. Use `argparse` with `--skip-sign` and `--skip-scan` flags for CI/dev convenience.

    h. Add `if __name__ == "__main__":` guard.

    i. Do NOT use `print()` from inside the frozen bundle — this script runs from the dev environment only, so print is fine here.
