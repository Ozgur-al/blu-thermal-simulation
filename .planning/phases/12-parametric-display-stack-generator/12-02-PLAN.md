    **Create `thermal_sim/ui/stack_generator_wizard.py`:**

    Use a custom `QDialog` with `QStackedWidget` + manual Back/Next/Generate/Cancel navigation (NOT `QWizard`) because the live 3D preview panel on the right requires a persistent side widget that QWizard cannot natively support (documented in RESEARCH Open Question 1). This is driven by the locked decision for "live 3D preview on the right side of the wizard."

    **Dialog layout:**
    ```
    QDialog
    └── QSplitter (horizontal)
        ├── Left QWidget (60% width)
        │   ├── QLabel (page title + step indicator "Step 2 of 6")
        │   ├── QStackedWidget (pages)
        │   └── QHBoxLayout (Back, Next, Generate, Cancel buttons)
        └── Right: Voxel3DView (40% width, live preview)
    ```

    Resize to 1200x750. Title: "Generate Display Stack".

    **Pages (QWidget subclasses, NOT QWizardPage since we use QDialog):**

    Each page class stores its data in instance attributes. The wizard collects all data via page accessors when generating.

    **Page 0 — ArchitecturePage:**
    - QComboBox: "ELED (Edge-Lit)" / "DLED (Direct-Lit)"
    - Brief description label below that updates with selection

    **Page 1 — PanelDimsPage:**
    - QDoubleSpinBox: Panel Width (mm), range 50-3000, default 450, suffix " mm"
    - QDoubleSpinBox: Panel Depth (mm), range 50-2000, default 300, suffix " mm"

    **Page 2 — LEDConfigPage:**
    - Uses QStackedWidget internally to show ELED or DLED sub-page based on architecture selection.
    - **ELED sub-page:**
      - 4 QCheckBox: Left, Right, Top, Bottom
      - QTableWidget (per enabled edge): columns = Edge (read-only), Count, LED Width (mm), LED Depth (mm), LED Height (mm), Power/LED (W), Margin (mm)
      - Defaults: count=10, led_w=0.6mm, led_d=4mm, led_h=1.4mm, power=0.3W, margin=12mm
      - When checkbox toggled, add/remove row from table
    - **DLED sub-page:**
      - QSpinBox: Rows (1-50, default 4), Columns (1-50, default 6)
      - QDoubleSpinBox: X Pitch (mm, default 60), Y Pitch (mm, default 60)
      - QDoubleSpinBox: X Offset (mm, default 30), Y Offset (mm, default 30)
      - QDoubleSpinBox: LED Width (mm, default 3), LED Depth (mm, default 3), LED Height (mm, default 1)
      - QDoubleSpinBox: Power per LED (W, default 0.5)

    **Page 3 — LayerThicknessPage:**
    - Uses QStackedWidget for ELED / DLED.
    - **Shared fields:** Panel thickness (mm), Reflector thickness (mm), Frame back plate (mm), Frame wall thickness (mm), Back cover (mm)
    - **ELED-specific:** LGP thickness (mm)
    - **DLED-specific:** Diffuser thickness (mm), Air cavity height (mm), PCB thickness (mm)
    - **Optical films section (shared):** QSpinBox for film count (0-10, default 0), QTableWidget with columns: Film #, Material (QComboBox from builtins), Thickness (mm). Rows auto-adjust with count spinner.
    - **Material dropdowns** per layer: each layer gets a QComboBox populated from builtin materials. Use `_make_material_combo()` pattern: iterate `load_builtin_library()` keys, set current to the default material for that layer.
    - All defaults from EledParams/DledParams defaults (converted to mm for display).

    **Page 4 — BoundaryConditionsPage:**
    - QDoubleSpinBox: Ambient temperature (C, default 25)
    - QCheckBox: Include radiation (default True)
    - QTableWidget with 6 rows (Top, Bottom, Front, Back, Left, Right):
      - Column 0: Face (read-only)
      - Column 1: Preset QComboBox ("Natural convection", "Forced air", "Enclosed still air", "Custom")
      - Column 2: h override (QDoubleSpinBox, enabled only when "Custom" selected, default 8.0)
    - Preset h values: Natural=8.0, Forced=25.0, Enclosed=3.0

    **Page 5 — MeshConfigPage:**
    - QDoubleSpinBox: Max Cell Size (mm), range 0.1-20, default 2.0, suffix " mm"
    - QSpinBox: Cells per Interval, range 1-10, default 1
    - QLabel: Estimated cell count (nice-to-have; compute from panel_w/panel_d/max_cell_size approximation. Show "~{n} cells" as an estimate.)

    **Navigation logic:**
    - `_go_back()`: decrement page index, update stack, update button visibility
    - `_go_next()`: increment page index, update stack, update button visibility, trigger `_update_preview()`
    - Back hidden on page 0; Next hidden on last page; Generate visible only on last page
    - On architecture change (page 0), update LED config page and layer thickness page to show correct sub-stack

    **Live preview:**
    - `_update_preview()` called on page transitions (not on every spinbox change to avoid performance issues per RESEARCH anti-pattern).
    - Wraps the generate call in try/except — skips preview on incomplete/invalid data (per RESEARCH Pitfall 5).
    - Calls `self._preview.update_structure(project)` with the partial/full VoxelProject.

    **Data collection — `_collect_eled_params() -> EledParams` / `_collect_dled_params() -> DledParams`:**
    - Read all page widgets, convert mm -> metres using `_m()`.
    - Build EdgeLedConfig objects from LED table rows.
    - Build OpticalFilm list from films table.
    - Map preset names to h values for BCs.

    **`_on_finish()`:**
    - Collect params, call `generate_eled()` or `generate_dled()`.
    - Store result in `self._generated_project`.
    - On error: show QMessageBox.critical, do NOT accept dialog.
    - On success: call `self.accept()`.

    New method `_open_stack_generator()`:
    ```python
    def _open_stack_generator(self) -> None:
        from thermal_sim.ui.stack_generator_wizard import StackGeneratorWizard
        wizard = StackGeneratorWizard(self._materials, parent=self)
        if wizard.exec() == QDialog.DialogCode.Accepted:
            project = wizard.generated_project()
            if project is not None:
                # Merge generated materials into library
                self._materials.update(project.materials)
                self._populate_mat_table()
                self._block_editor.update_material_list(self._materials)
                # Load project into editor (replaces all blocks)
                self._block_editor.load_project(project)
                self._update_3d_structure()
    ```
