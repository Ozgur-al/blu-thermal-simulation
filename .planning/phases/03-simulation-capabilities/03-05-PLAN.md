    2. Add test_single_node_rc_with_square_wave_power_matches_analytical():
       - Material: k=10, density=2700, cp=900
       - Layer: 0.002m thick, full area
       - Boundaries: h_top=15, h_bottom=0, side=0, ambient=25C
       - Power profile: [(0, 2.0), (0.5, 0.0), (1.0, 2.0)] — 0.5s on, 0.5s off, repeating
       - Transient: dt=0.01s, total=2.0s, output_every=0.01s
       - Initial temp: 25C
       - Hand calc: tau = (density * cp * thickness * area) / (h_top * area) = rho*cp*t/h
         - During ON: T_ss = T_amb + Q/(h*A), T(t) = T_ss + (T0 - T_ss)*exp(-t/tau)
         - During OFF: T_ss = T_amb, T(t) = T_ss + (T0 - T_ss)*exp(-t/tau)
         - Compute expected T at t=0.5, 1.0, 1.5, 2.0 analytically
       - Assert temperatures at end of each half-period within rel_tol=0.01

    3. Add test_two_node_lateral_spreading_matches_hand_calculation():
       - Material: k_in_plane=50, k_through=50, density=2000, cp=900
       - Layer: 0.003m thick
       - Panel: width=0.04m, height=0.02m
       - Mesh: 2x1 (two cells in x, one in y)
       - Heat source: rectangle, layer="L", power=0.5W, x=0.005, y=0.005, width=0.019, height=0.019 (covers left cell only)
       - Boundaries: h_top=10, h_bottom=0, side=0, ambient=25C
       - Hand calc:
         - Cell area = (width/2) * height = 0.02 * 0.02
         - dx = width / 2 = 0.02
         - dy = height / 1 = 0.02
         - g_top_per_cell = h_top * cell_area
         - g_lat = k_in_plane * thickness * dy / dx
         - Node 0 (left, has power): (g_top + g_lat)*T0 - g_lat*T1 = Q + g_top*T_amb
         - Node 1 (right, no power): -g_lat*T0 + (g_top + g_lat)*T1 = g_top*T_amb
         - Solve 2x2 system
       - Assert both temperatures within rel_tol=1e-6
