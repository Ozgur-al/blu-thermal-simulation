<objective>
Build the parametric sweep engine backend and wire it into the CLI. The sweep engine iterates over a list of parameter values, deep-copies the project for each run, applies the parameter mutation, solves, and collects per-layer summary stats (T_max, T_avg). CLI users can invoke sweeps via `--sweep sweep.json`.

Purpose: SIM-01 requires the user to define a parametric sweep and execute multiple solver runs. This plan builds the entire backend and CLI surface. GUI wiring is deferred to 03-04.
Output: SweepConfig/SweepResult dataclasses, SweepEngine with progress callback, CLI --sweep flag, unit tests.
</objective>
