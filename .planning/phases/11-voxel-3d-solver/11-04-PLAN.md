    **Voxel3DView** (`thermal_sim/ui/voxel_3d_view.py`):
    Create a QWidget wrapping `pyvistaqt.QtInteractor` (follow the existing `assembly_3d.py` pattern).

    Features:
    1. **Structure view**: Show assembly blocks as color-coded `pv.Box` meshes. Use `_material_color_map` pattern from existing code for consistent material colors. Block labels displayed as text annotations.

    2. **Temperature overlay**: After solving, create a `pv.RectilinearGrid(x_edges, y_edges, z_edges)` with `cell_data['Temperature_C']` from the result array. Use `.ravel(order='F')` for VTK Fortran-order convention.

    3. **Slice planes**: Three slider widgets (one per axis: X, Y, Z) that create cutting planes through the RectilinearGrid to reveal internal temperatures. Use `grid.slice(normal='x', origin=...)` pattern.

    4. **Block transparency/hide**: QListWidget or checkboxes listing all blocks. User can toggle visibility or set transparency per block. When toggling, add/remove the corresponding Box actor.

    5. **Temperature threshold filter**: QDoubleSpinBox + QPushButton. Applies `grid.threshold(value, scalars='Temperature_C')` to show only voxels above the threshold.

    6. **Probe markers**: Labeled point actors at probe (x, y, z) positions. Show temperature readout as annotation text.

    7. **Toolbar**: Buttons for: Reset Camera, Toggle Structure/Temperature, X/Y/Z Slice toggles, Threshold toggle.

    Methods:
    - `update_structure(project: VoxelProject)` — rebuild block visualization
    - `update_temperature(result: VoxelSteadyStateResult | VoxelTransientResult, timestep=None)` — overlay temperature data
    - `clear()` — reset view
