**Step 2 — Refactor BC application in voxel_network_builder.py:**

Replace the current Step 4 logic (lines 207-280) which uses only `project.boundary_groups[0]` for all faces. Instead:

1. Define a helper `_face_matches_group(face_name: str, group: BoundaryGroup) -> bool` that returns True if `"all"` in group.faces or face_name in group.faces.

2. For each of the 6 grid-boundary faces (bottom, top, front, back, left, right), iterate `project.boundary_groups` and find the FIRST group whose `faces` list matches. If no group matches, skip that face (no BC applied).

3. Apply the matched group's BC to that face using the existing `_apply_face_bc` logic.

The refactored structure:
```python
_FACE_SPECS = [
    ("bottom", lambda ny, nx, nz, dy, dx, dz: (
        np.zeros((ny, nx), dtype=int),  # iz
        *np.meshgrid(np.arange(ny), np.arange(nx), indexing='ij'),  # iy, ix
        np.outer(dy, dx),  # face_area
    )),
    ("top", ...),
    ("front", ...),
    ("back", ...),
    ("left", ...),
    ("right", ...),
]

for face_name, make_arrays in _FACE_SPECS:
    # Find first matching boundary group
    bc_group = None
    for bg in project.boundary_groups:
        if "all" in bg.faces or face_name in bg.faces:
            bc_group = bg
            break
    if bc_group is None:
        continue
    iz_arr, iy_arr, ix_arr, fa_arr = make_arrays(ny, nx, nz, dy, dx, dz)
    _apply_face_bc(bc_group.boundary, iz_arr, iy_arr, ix_arr, fa_arr)
```

Move `_apply_face_bc` to accept a `SurfaceBoundary` parameter instead of capturing `bc` from outer scope. This way each face can use a different boundary condition.
