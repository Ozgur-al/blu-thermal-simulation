    ```python
    @dataclass
    class VoxelThermalNetwork:
        a_matrix: scipy.sparse.csr_matrix  # conductance matrix (n_cells x n_cells)
        b_vector: np.ndarray               # forcing vector (n_cells,)
        c_vector: np.ndarray               # thermal capacity vector (n_cells,)
        mesh: ConformalMesh3D
        material_grid: np.ndarray          # (nz, ny, nx) material names
    ```

    `def build_voxel_network(project: VoxelProject) -> VoxelThermalNetwork`:

    **Step 1: Build conformal mesh and assign materials.**
    Call `build_conformal_mesh(project.blocks, project.mesh_config.cells_per_interval)` then `assign_voxel_materials(mesh, project.blocks)`.

    **Step 2: Build material property arrays.**
    Create (nz, ny, nx) arrays for k_in_plane, k_through, density, specific_heat, emissivity from material_grid + project.materials dict.

    **Step 3: Assemble conductance matrix (vectorized COO).**
    For each axis direction (x, y, z), compute conductance between all neighbor pairs:
    - x-direction: face area = dy[j] * dz[k], half-distances = dx[i]/2 and dx[i+1]/2
    - y-direction: face area = dx[i] * dz[k], half-distances = dy[j]/2 and dy[j+1]/2
    - z-direction: face area = dx[i] * dy[j], half-distances = dz[k]/2 and dz[k+1]/2
    - Use k_in_plane for x and y directions, k_through for z direction
    - Conductance G = face_area / (half_1/k_1 + half_2/k_2) — harmonic mean
    - Vectorize using NumPy: build arrays of all neighbor pairs per axis, compute G in one pass, accumulate COO triplets
    - Add -G to off-diagonals, +G to both diagonals

    **Step 4: Auto-detect exposed faces and apply boundary conditions.**
    An exposed face is: (a) a face at the grid boundary (ix=0, ix=nx-1, etc.), OR (b) a face between a non-air voxel and an air voxel.
    For each boundary group in the project, apply convection + optional radiation:
    - `G_conv = h * face_area`
    - If include_radiation: `h_rad = 4 * eps * STEFAN_BOLTZMANN * (T_amb_K)^3`, `G_rad = h_rad * face_area`
    - `G_surface = G_conv + G_rad`
    - Add to diagonal, add `G_surface * T_amb` to b_vector
    - For now, assign ALL exposed faces to the first boundary group. Future enhancement: named group assignment.

    **Step 5: Build heat source vector.**
    For each SurfaceSource, find the block it references, determine which voxels are on that face, apply power distribution (uniform over face area, or shaped mask for rectangle/circle).

    **Step 6: Build capacity vector.**
    `C[i] = density * specific_heat * volume` where volume = dx[ix] * dy[iy] * dz[iz].

    Convert COO to CSR. Return VoxelThermalNetwork.

    **Key implementation note:** Use vectorized NumPy operations throughout. NO Python for-loops over individual voxels. The existing `_add_link_vectorized` pattern from network_builder.py is the template — adapt for 3D non-uniform grids.
