Output: Zone sub-table with inline preview in Layers tab, dashed zone overlay on temperature map.

Each entry is a list of dicts: `{"material": str, "x": float, "y": float, "width": float, "height": float}` (SI metres). Populate from `Layer.zones` in `_populate_ui_from_project()`.

4. **Zone table interaction**:
   - "Add Zone" button: Inserts a new row with empty values (default material from first project material, x=0, y=0, width=panel_width, height=panel_height).
   - "Remove Zone" button: Removes the selected row from the zone table and `_layer_zones[row]`.
   - On `zone_table.cellChanged`: Call `_on_zone_table_changed()` which reads all rows, updates `_layer_zones[layer_row]`, and calls `_refresh_zone_preview(layer_row)`.

5. **Zone preview canvas**: Create `self._zone_preview_canvas = MplCanvas(width=3.5, height=2.5, dpi=80)` following the existing MplCanvas pattern. Add it to the zone panel layout.

6. **`_refresh_zone_preview(layer_row)`**: Clear the preview canvas axes. Draw the layer footprint as a light gray rectangle (0 to width_mm, 0 to height_mm). For each zone in `_layer_zones[layer_row]`, draw a `matplotlib.patches.Rectangle` with dashed edges (white or colored by material), and a text label at the zone corner. Call `canvas.draw_idle()`.

   ```python
   zones = [MaterialZone(material=z["material"], x=z["x"], y=z["y"],

            for z in self._layer_zones.get(row, [])]
   ```
   Pass `zones=zones` to the Layer constructor.

8. **Handle layer add/remove**: When layers are added/removed from the table, update `_layer_zones` accordingly. On add: `_layer_zones[new_row] = []`. On remove: shift keys down.

9. **Zone table coordinate display**: All coordinates in the zone table are displayed in mm. When reading from `_layer_zones` (SI metres), multiply by 1000 for display. When writing back, divide by 1000.

**Anti-patterns to avoid:**

   ```python
   if zones:
       from matplotlib.patches import Rectangle
       for zone in zones:
           x_mm = zone.x * 1000.0
           y_mm = zone.y * 1000.0
           w_mm = zone.width * 1000.0
           h_mm = zone.height * 1000.0
           rect = Rectangle(
               (x_mm, y_mm), w_mm, h_mm,
               linewidth=1.2,
               edgecolor="white",
               linestyle="--",
               facecolor="none",
               zorder=6,
           )
           ax.add_patch(rect)
           ax.text(
               x_mm + 0.5, y_mm + 0.5,
               zone.material,
               fontsize=6, color="white",
               va="bottom", ha="left",
               zorder=7,
           )
   ```

   ```python
   # Determine current layer name (strip z-suffix if present)
   ```
