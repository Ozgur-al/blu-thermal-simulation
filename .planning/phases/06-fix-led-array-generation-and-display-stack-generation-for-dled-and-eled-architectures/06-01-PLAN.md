Two public functions:

```python
def dled_template(
    panel_width: float,    # metres
    panel_height: float,   # metres
```

```python
def eled_template(

```

Both return dict with keys: "layers" (list[Layer]), "materials" (dict[str, Material]), "led_arrays" (list[LEDArray]), "boundaries" (BoundaryConditions).

DLED template layers (bottom to top, all thicknesses in metres):
- Back Cover: Aluminum, 0.0008 (0.8 mm)
- Metal Frame: Steel, 0.001 (1.0 mm)
- LED Board: FR4, 0.001 (1.0 mm)
- Diffuser: PC, 0.002 (2.0 mm) — first optical sheet
- BEF: PC, 0.0003 (0.3 mm) — second optical sheet
- (Additional optical sheets named "Optical Sheet N" using PC, 0.0003 each, for optical_layers > 2)
- OCA: OCA, 0.00015 (0.15 mm)
- Display Cell: Glass, 0.0011 (1.1 mm)
- Cover Glass: Glass, 0.0018 (1.8 mm)

DLED LED array defaults: mode="grid", layer="LED Board", count_x=8, count_y=6, pitch derived from panel size (panel_width / (count_x + 1) and panel_height / (count_y + 1)), power_per_led_w=0.5, footprint_shape="rectangle", led_width=0.003, led_height=0.003, zone_count_x=1, zone_count_y=1. Set panel_width and panel_height on the LEDArray. Set offset_left=offset_right=offset_top=offset_bottom to 10% of panel dimensions.

ELED template layers: Same as DLED but replace "LED Board" (FR4) with "LGP" (PMMA, 0.004 thickness). Per RESEARCH.md recommendation, edge LEDs are placed ON the LGP layer (option 1 — the 2.5D model approximation).

ELED LED array defaults: mode="edge", layer="LGP", edge_config=parameter, count_x=20 (for bottom/top edges), count_y=15 (for left/right edges), pitch_x derived from panel_width / (count_x + 1), pitch_y derived from panel_height / (count_y + 1), power_per_led_w=0.3, edge_offset=0.005, footprint_shape="rectangle", led_width=0.002, led_height=0.001. Set panel_width and panel_height.

Both templates: boundaries with side.convection_h=25.0 (metal frame enhancement per RESEARCH.md), top/bottom default SurfaceBoundary().

Materials: Load from load_builtin_library(). Filter to only materials referenced by template layers. This avoids shipping unused materials.

Write tests FIRST in tests/test_stack_templates.py, then implement.
