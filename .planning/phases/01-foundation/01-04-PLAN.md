  <how-to-verify>
Launch the GUI: `python -m thermal_sim.app.gui`

**1. Undo/Redo (GUI-02):**
   - Load a project (File > Open or use the startup project)
   - Edit a cell in the Materials table (e.g., change a conductivity value)
   - Press Ctrl+Z — the cell should revert to its previous value
   - Press Ctrl+Y — the cell should change back to the edited value
   - Verify Edit menu shows "Undo" and "Redo" with correct action text

**2. Simulation with Progress (GUI-03):**
   - Switch mode to "transient" in the toolbar dropdown
   - Press F5 (or Run menu > Run Simulation)
   - Observe the progress bar updating in the center of the status bar
   - Observe timestep count and peak temperature updating
   - Start another transient run, then press Escape (or Run menu > Cancel)
   - Confirm the status bar shows "Cancelled" and the Run button re-enables

**3. Status Bar (GUI-04):**
   - Check left zone shows file path (or "No file")
   - Check center zone shows solver state
   - Check right zone shows last run time and mesh size after a run
   - After a successful run, verify T_max, solve time, and mesh dimensions are displayed

**4. Window Title (GUI-05):**
   - With a clean project, verify no asterisk in title
   - Edit a cell — asterisk should appear in title
   - Press Ctrl+Z until all changes are undone — asterisk should disappear
   - Edit a cell again, then Ctrl+S to save — asterisk should disappear

**5. CLI Parity (GUI-06):**
   - Run menu > Set Output Directory — folder browser opens, select a folder
   - Run menu > Export CSV — CSVs are exported to the output directory (run a simulation first)
   - Toolbar mode dropdown switches between steady and transient

**6. Keyboard Shortcuts (GUI-07):**
   - F5 runs simulation from any focused widget
   - Escape cancels running simulation
   - Ctrl+S saves project (shows Save dialog if no path set)
   - Ctrl+Z / Ctrl+Y undo/redo

**7. Unsaved Changes Protection:**
   - Edit a cell (asterisk appears)
   - Try to close the window — Save/Discard/Cancel dialog should appear
   - Click Cancel — window stays open
   - Click Discard — window closes
   - Re-open, edit, close again — click Save — save dialog appears
