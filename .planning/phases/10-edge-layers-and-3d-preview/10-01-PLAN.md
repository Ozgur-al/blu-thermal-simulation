   ```python
   @dataclass(frozen=True)
   class EdgeLayer:
       """One lateral layer slice on an edge (material + thickness in metres)."""
       material: str
       thickness: float

       def __post_init__(self) -> None:
           if not self.material.strip():
               raise ValueError("EdgeLayer material must not be empty.")
           if self.thickness <= 0.0:
               raise ValueError(f"EdgeLayer thickness must be > 0, got {self.thickness}.")

       def to_dict(self) -> dict:
           return {"material": self.material, "thickness": self.thickness}

       @classmethod
       def from_dict(cls, data: dict) -> "EdgeLayer":
           return cls(material=data["material"], thickness=float(data["thickness"]))
   ```

  <action>
1. **Add generate_edge_zones() to stack_templates.py**:
   ```python
   def generate_edge_zones(
       layer: "Layer",
       panel_width: float,
       panel_height: float,
   ) -> list[MaterialZone]:
   ```
   Import `MaterialZone` at the top of the function body (lazy import to avoid circular).

   **Algorithm** (per RESEARCH.md Pattern 3):
   - Extract bottom/top/left/right lists from `layer.edge_layers` (default to empty).
   - Compute `bottom_total`, `top_total`, `left_total`, `right_total` thickness sums.
   - Validate: `left_total + right_total < panel_width` and `bottom_total + top_total < panel_height`. Raise `ValueError` with descriptive message if violated.
   - If `edge_layers` is empty or all edges are empty, return `[]`.
   - **Bottom zones**: Full-width (panel_width) strips stacked from y=0 upward. Each zone: `MaterialZone(material=el.material, x=panel_width/2, y=y + el.thickness/2, width=panel_width, height=el.thickness)`. Increment y by el.thickness.
   - **Top zones**: Full-width strips stacked from y=panel_height downward. Each zone: `MaterialZone(material=el.material, x=panel_width/2, y=y - el.thickness/2, width=panel_width, height=el.thickness)`. Decrement y by el.thickness.
   - **Left zones**: Interior-height strips from x=0 rightward. Interior y range: `[bottom_total, panel_height - top_total]`. Interior height = `panel_height - bottom_total - top_total`. Each zone: `MaterialZone(material=el.material, x=x + el.thickness/2, y=bottom_total + interior_height/2, width=el.thickness, height=interior_height)`. Increment x.
   - **Right zones**: Interior-height strips from x=panel_width leftward. Each zone: `MaterialZone(material=el.material, x=x - el.thickness/2, y=bottom_total + interior_height/2, width=el.thickness, height=interior_height)`. Decrement x.

   **CRITICAL**: `MaterialZone` uses CENTER coordinates (x, y are center, not corner). The rasterizer computes corners as `zone.x - zone.width/2`.

2. **Wire solver**: In `network_builder.py`, at the zone rasterization loop (line ~255-258), replace the direct `_rasterize_zones(layer, ...)` call with edge-zone merging:
   ```python
   import dataclasses
   from thermal_sim.models.stack_templates import generate_edge_zones

   for l_idx, layer in enumerate(project.layers):
       if layer.edge_layers:
           edge_zones = generate_edge_zones(layer, project.width, project.height)
           effective_zones = edge_zones + list(layer.zones)
           layer_with_merged = dataclasses.replace(layer, zones=effective_zones)
           maps = _rasterize_zones(layer_with_merged, project.materials, grid)
       else:
           maps = _rasterize_zones(layer, project.materials, grid)
       zone_maps.append(maps)
   ```
   Edge zones are prepended so manual zones (appended last) win on overlap (last-defined-wins in rasterizer).
